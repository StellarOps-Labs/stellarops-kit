# Getting Started

## Installation

Install dependencies:

```bash
npm install
```

## Running locally

Start the development server:

```bash
npm run dev
```

Open `http://localhost:3000` and visit `/toolkit`.

## Testing

Run the test suite:

```bash
npm run test
```

Use watch mode during development:

```bash
npm run test:watch
```

## Building

Create a production build:

```bash
npm run build
```

Run linting and type checks before opening a pull request:

```bash
npm run lint
npm run typecheck
```
