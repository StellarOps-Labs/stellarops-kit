# Maintainer Readiness Checklist

- [ ] README explains the project and local setup.
- [ ] CONTRIBUTING explains branch, test, and PR expectations.
- [ ] CODE_OF_CONDUCT and LICENSE are present.
- [ ] CI runs lint, typecheck, tests, and build.
- [ ] Issue templates support bugs, features, and contributor tasks.
- [ ] Labels cover difficulty, domain, and triage status.
- [ ] Beginner issues include acceptance criteria.
- [ ] Stellar/Soroban tasks explain relevant ecosystem concepts.
