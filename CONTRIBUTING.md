# Contributing

Thanks for helping improve StellarOps Kit.

## Local setup

1. Install dependencies with `npm install`.
2. Start the app with `npm run dev`.
3. Run checks with `npm run lint`, `npm run typecheck`, and `npm run test`.

## Issue workflow

Choose an issue with acceptance criteria and confirm the expected behavior before opening a pull request. If the issue is unclear, ask for clarification instead of guessing.

## Pull requests

Keep pull requests focused. Include tests for reusable logic and visible UI behavior. Update docs when a workflow, template, or maintainer expectation changes.
